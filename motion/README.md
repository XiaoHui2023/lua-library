# motion

运动结果计算库，把位移数据、修正器和渲染器分开。

- `data` 统一描述起点、高度、位移、朝向和时间步长。
- modifier 负责修改运动数据。
- renderer 负责把计算结果应用到外部对象。

## 设计特性

### 数据归一

运动先被整理成统一数据结构，再进入修正和解析。直线、击退、跳跃等动作可以共享同一份位移字段。

### 修正链

modifier 只关心如何改运动数据。多个修正可以叠加，最终由 `resolve` 计算位置和高度。

### 渲染分离

库层只产出运动结果，不直接操作引擎对象。实际移动、特效或 UI 表现交给 renderer 或上层框架。

## 目录

```text
init.lua      # 运动数据、reset 与 resolve
modifier.lua  # 运动修正器
renderer.lua  # 运动结果渲染入口
test.lua      # 运动行为测试
```
