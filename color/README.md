# color

颜色对象与颜色文本工具，负责颜色定义、色板查找和文本渲染接入。

- 颜色对象只读，统一用 RGB 通道表达。
- 默认色板覆盖常用阵营色和黑白灰。
- 文本渲染函数可由运行时注入，库本身不绑定具体引擎。

## 设计特性

### 只读颜色

颜色由 `define` 或 `from_hex` 创建，创建后不能改通道值。这样常量色、色板色和业务传入色可以安全共享。

### 色板匹配

`nearest` 按颜色距离从候选色板中找最接近的颜色；未传候选时使用默认色板。`random` 也只从默认色板取值。

### 文本渲染

库只定义渲染入口，默认返回原文本。宿主可以注入颜色标签渲染函数，游戏侧不用关心具体文本格式。

## 目录

```text
init.lua  # 颜色对象、色板、文本模式与渲染入口
test.lua  # 颜色转换与渲染测试
```

## API

| 名称 | 用途 |
| --- | --- |
| `define` | 用 RGB 创建颜色 |
| `from_hex` / `to_hex` | 十六进制颜色互转 |
| `nearest` / `distance` / `hue` | 色板匹配与颜色计算 |
| `remove` | 移除文本中的颜色标记 |
| `set_renderer` / `reset_renderer` | 注入或恢复文本渲染函数 |
