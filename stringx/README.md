# stringx

字符串工具库，补充按键映射和文本布局相关处理。

- 常用字符串操作集中在库入口。
- `keymap` 维护输入按键名称映射。
- `text_layout` 处理文本尺寸和排版计算。

## 设计特性

### 文本辅助

通用字符串处理放在同一入口，业务侧不需要在各处复制 trim、split 或格式化逻辑。

### 输入映射

按键名称和显示文本放在 `keymap`，让 UI 提示与输入配置使用同一份映射。

### 布局计算

文本布局计算独立于具体 UI 框架。上层只接收结果，再决定如何渲染。

## 目录

```text
init.lua        # 字符串工具门面
keymap.lua      # 按键映射
text_layout.lua # 文本布局计算
test.lua        # 字符串工具测试
```
