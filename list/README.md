# list

数组式列表结构，适合需要稳定遍历、排序和安全删除的场景。

- 支持首尾增删、遍历和排序。
- 遍历中删除不会破坏当前遍历流程。
- 对象通过可调用表创建，使用成本接近普通 table。

## 设计特性

### 稳定遍历

列表把元素数量和内部数组封装起来，删除时维护索引关系。需要遍历同时修改集合时，比直接操作 table 更可控。

### 常用操作

列表提供 append、pop、remove、clear、sort 等集合操作。游戏对象集合、消息历史和临时队列都可以复用同一套行为。
